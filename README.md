# AirHub

A modern full-stack platform for exploring global airports and the connections between them.

AirHub provides an interactive map-based interface where users can discover airports, flight routes, distances, and travel patterns.  
It is designed as a portfolio-quality application showcasing modern API design, clean architecture, frontend‚Äìbackend synchronization, and scalable infrastructure.

---

## üåê What AirHub Does

- Visualizes **all global airports** and their metadata
- Shows **routes and connections** between airports
- Allows searching airports by name, city, IATA code, and country
- Offers a fast, modern API built with FastAPI
- Uses PostgreSQL for structured airport data
- Provides a clean and minimal UI inspired by flight-connection applications
- Can be deployed online and extended with user features, filters, analytics, etc.
